# Faculty Load Repo
This repo contains all official sql scripts pertaining to faculty load.

### faulty_load
This script pulls
- D-number
- first name
- last name
- instructor status: adjunct, tenure, tenure-track, non-tenure-track, part-time faculty
- percent responsibility: percent responsibity for the course
- overload indicator
- term code
- course record number
- course number
- section number
- course title
- college 
- department
- total enrolled credit hours:
